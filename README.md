## 📌 Project Overview

This repository contains the source code for our final course project. The objective is to apply Python programming, Object-Oriented paradigms, and advanced data structures to solve real-world problems. We have developed a comprehensive **Library Management System**, split into two distinct tasks:

### 🚀 [Task 1: OOP-Based Library Management System](./Task1_LibrarySystem/)
A fully functional, console-based application demonstrating advanced **Object-Oriented Programming (OOP)** concepts taught in the course.
*   **Key Features:** Modular design (4 independent `.py` modules), Encapsulation (protected attributes), Inheritance & Polymorphism (Student vs. Faculty members), and Abstraction (using the `ABC` module).
*   **Directory:** [`/Task1_LibrarySystem`](./Task1_LibrarySystem/) - *Click to view code and detailed Task 1 documentation.*

### 🧠 [Task 2: Self-Study Priority Waitlist & Sorting](./Task2_PriorityWaitlist/)
An independent research and implementation of topics **NOT covered in the course curriculum**, designed to enhance the library system's efficiency.
*   **New Data Structure:** **Max-Heap (Priority Queue)** implemented from scratch to handle book waitlists, ensuring users with higher institutional priority (e.g., Faculty) are served first.
*   **New Algorithm:** **Heap Sort**, an in-place $O(n \log n)$ algorithm used to rank the entire library catalog based on book popularity.
*   **Directory:** [`/Task2_PriorityWaitlist`](./Task2_PriorityWaitlist/) - *Click to view code, algorithmic complexity analysis, and demo scripts.*
